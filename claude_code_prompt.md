# Claude Code 개선 지시문

이 폴더의 `index.html`은 국회 관련 토론회·정책세미나 정보를 주기적으로 조회하기 위한 웹페이지 초안입니다. 목적은 공공데이터포털의 `국회 국회사무처_국회의원 세미나 일정` API를 활용해 앞으로 2주 이내 세미나를 매일 아침 확인하고, 관심 키워드에 맞는 일정만 표 또는 짧은 브리핑으로 보여주는 것입니다.

## 현재 요구사항

- 검색 대상: 앞으로 2주 이내 국회의원 정책세미나·토론회
- 필터 키워드: 재정, 예산, 산업정책, 저출산, 연금, 복지, AI, 지역균형, 문화, 예술, 플랫폼, 디지털
- 추출 항목: 제목, 일시, 장소, 주최 의원, 공동주최 기관, 관련 위원회, 링크
- 처리 방식: API 조회 → 2주 이내 필터 → 키워드 필터 → 중복 제거 → 중요도 점수 부여 → 요약 출력
- 결과 형식: 표 보기와 짧은 브리핑 보기

## 우선 개선할 것

1. 실제 API 응답 구조를 확인해 `normalizeEvent()`의 필드 매핑을 정확하게 고쳐라.
2. 브라우저 CORS 문제가 있으면, 작은 Node.js 또는 Python 백엔드 프록시 구조로 분리하라.
3. API 키는 클라이언트 HTML에 저장하지 말고 서버 환경변수 `ASSEMBLY_API_KEY`로 읽도록 바꿔라.
4. 매일 아침 8시에 자동 조회하는 구조를 제안하고, 가능하면 실행 가능한 스크립트까지 만들어라.
5. 결과를 마크다운 파일, 이메일, 또는 텔레그램 메시지로 내보낼 수 있게 설계하라.
6. 디자인은 현재의 차분한 운영 도구 느낌을 유지하되, 표의 가독성과 모바일 보기를 개선하라.
